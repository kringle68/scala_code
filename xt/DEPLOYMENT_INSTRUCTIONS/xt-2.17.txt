===============================
XT 2.17 Deployment Instructions
===============================

Release Date:       2010/06/1


These instructions are meant to be used in conjuction with those detailed in confluence:
    http://confluence.net-a-porter.com/display/DEV/XT+Deployment#XTDeployment-XTApplicationdeployment

* Configuration files in ``env-conf/*_live/`` have been changed to be
  up to date w.r.t. actual live config, please check

* WSDL files in ``env-conf/*_live/payment-wsdl/`` have been updated,
  taking them from http://psp.gs2.nap:8080/psp/psp - please check that
  they are correct

* check that the data patch mentioned in http://jira/browse/REL-147 is applied

* DC1 - the format of the exported file for sending premier routing
  data to DHL has changed. The recipent of this file may need to be
  notified.

* *BEFORE* deployment, stop the ActiveMQ consumer server
  (``xt_activemq_dc_daemon.pl``)

  Then install the new code

  Then restart the consumer server.

  This is needed because one of the libraries the consumer depends
  upon has been updated to fix REL-200
  

*** IMPORTANT ***

Please remember to apply DB patches for both DCs


X. Lastly
---------

Paste any console output from the steps above as a

  {noformat}
    [replace with console output]
  {noformat}

comment to the JIRA ticket for this release.
